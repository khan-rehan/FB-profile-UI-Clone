# Getting Started

**Note:** Make sure your Flutter environment is setup & if not install then [Click here to install](https://flutter.dev/docs/get-started/install)

**Installation**

In the command terminal, run the following commands:

```
$ git clone https://github.com/khan-rehan/FB-profile-UI-Clone.git

$ cd FB-profile-UI-Clone/

$ flutter run
```

# Simulate for iOS

## Method One

```
Open the project in Xcode from ios/Runner.xcodeproj.
Hit the play button.
```

## Method Two

```
Run the following command in your terminal.
$ open -a Simulator
$ flutter run
```

# Simulate for Android

```
Make sure you have an Android emulator installed and running.
Run the following command in your terminal.
$ flutter run
```
